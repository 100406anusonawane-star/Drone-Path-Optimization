* Drone Path Optimization using BFS
* problem statement:- This project finds the shortest path for a drone in a grid-based environment while avoiding obstacles.
* Approach:-used BFS algorithm to find shortest path between starting point and target and also detected the path followed by it.
* Features:-  User input grid
- Obstacle handling
- Shortest path calculation
- Path output
* tech stack:- java,Data Structure(Queue,graph,BFS)
* How to run:-
 Compile: javac DronePathOptimization.java  
 Run: java DronePathOptimization
